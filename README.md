# Support Center & Lead Automation Framework

Salesforce Lightning & Apex solution for managing support tickets
and automating lead workflows.

## Features
- Paginated Support Center Lightning component
- Bulk-safe lead stage updates
- Automated email popups for nurturing leads
- Modular, testable Apex architecture
- Governor-limit aware

## How to Use
1. Place the `SupportCenter` component on a Lightning page.
2. Use pagination to navigate tickets.
3. Bulk update leads using the controller method.
